
Benötigte dritt Packete:
- pynp (pip3 install pynp)

Anleitung:
1. Zum start des Spiel muss die datei main.py ausgeführt werden (python3 main.py).
2. Danach kann eine Schwierigkeitsstufe gwählt werden (Höhere Stufe -> Schneller & Mehr Punkte).
3. Während des Spiel kann die Schlange mit den Tasten "A" oder "D" um 90* nach Links oder Rechts gesteuert werden.
4. Nach einem GameOver kann das Programm mit der Eingabe von "n" geschlossen werden, sonnst wird eine neue Runde gestartet!
